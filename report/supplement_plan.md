## 二、补充 SWD 或 Chamfer 指标

### 需要补什么

在现有指标基础上增加至少一个几何距离指标：

- `SWD`：Sliced Wasserstein Distance，适合二维分布，计算稳定，和对方报告一致。
- `Chamfer Distance`：点云几何距离，直观反映支撑集形状差异。

建议最终指标体系为：

```text
MMD + SWD/Wasserstein + Chamfer + Coverage + Precision
```

如果担心指标过多，可以保留：

```text
MMD, SWD, Coverage, Precision
```

### 为什么需要补

我们的 Coverage/Precision 很适合诊断模式覆盖与样本真实性，但和对方相比，缺少一个更标准的几何距离指标。加入 SWD 或 Chamfer 后，指标体系会更完整，避免老师觉得评价维度偏少。

### 需要产出的表图

1. 更新 `表 6.1`，加入 SWD 或 Chamfer 列。
2. 更新 `metrics_heatmap.pdf`，加入新增指标。
3. 更新综合得分计算方式，说明各指标归一化后等权或按类别加权。

### 报告中如何修改

位置：第六节“生成质量评价”。

新增一个小节：

```text
6.x 切片 Wasserstein 距离 / Chamfer 距离
```

并说明它与现有指标互补：MMD 偏全局分布匹配，SWD/Chamfer 偏几何支撑集，Coverage/Precision 偏样本级诊断。

## 三、拓展鲁棒性实验需要扩展

### 需要补什么

当前鲁棒性实验主要在 Spiral 上做均匀噪声污染。建议扩展为：

```text
分布：Ring, Spiral
污染比例：0%, 5%, 10%
污染类型：
1. 均匀噪声：[-4,4]^2 内随机散点
2. 局部团簇噪声：在真实支撑集外生成一个小异常团簇
3. 边界噪声：沿分布外围或中心空洞附近添加干扰点
```

### 为什么需要补

对方报告的鲁棒性部分覆盖了多种污染类型。我们的鲁棒性实验结论“VAE 在噪声污染下更鲁棒”很有意思，但如果只在 Spiral + 均匀噪声上验证，支撑不够强。

### 需要产出的表图

1. 新增鲁棒性总表：按污染类型和比例给出两类模型均值指标。
2. 新增鲁棒性曲线：横轴为污染比例，纵轴为 MMD / Coverage / Precision。
3. 新增代表性可视化：展示污染训练集、VAE 生成结果、DDPM 生成结果。

### 报告中如何修改

位置：第八节“拓展实验”中的鲁棒性分析。

结论需要更克制：

```text
VAE 的 KL 正则在无结构噪声下表现出一定去噪效果；但在结构化团簇污染下，两类模型都可能学习到异常模式，鲁棒性结论依赖污染类型。
```

## 六、报告结论需要更克制、更像实验论文

### 当前风险

当前报告里有一些表述偏强，例如：

```text
DDPM 在连续流形分布上具有更好的覆盖性和局部几何保持能力
```

这个结论大体成立，但如果补充多种子和更多指标后，可能出现某些指标上 VAE 更优。需要避免过度宣称。

### 建议改法

将结论写成条件化表达：

```text

```
